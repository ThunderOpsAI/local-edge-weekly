# Milestones

## M1: Contract and Foundation

- Create the schema and industry profile artifacts.
- Stand up auth, accounts, projects, and targets.
- Define the report contract and diagnostics contract.
- Seed trial credits and account roles.

## M2: Engine and API

- Split the analysis engine into stage-based modules.
- Introduce queued runs and checkpoints.
- Expose API routes for projects, runs, reports, and diagnostics.
- Persist versioned reports and canonical signals.

## M3: Dashboard and Trends

- Ship the dashboard shell for project setup and latest reports.
- Add diagnostics and run history.
- Add trend deltas once at least two runs exist.
- Keep the UI focused on trust, clarity, and actionability.
